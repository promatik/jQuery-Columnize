jQuery Columnize Demo

 

$().columnize()

$('.lorem-ipsum').columnize({
width:210,
height:300
});

This function columnize the element selected and return an Int with the number of columns generated.

$().columnize('getColumns')

$('.lorem-ipsum').columnize('getColumns', {
width:210,
height:300
});

This function returns the number of columns that text will fill after "columnize".
